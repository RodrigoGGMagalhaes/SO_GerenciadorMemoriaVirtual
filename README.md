# SO_GerenciadorMemoriaVirtual

## Descrição

Nesse projeto iremos finalizar um código em linguagem C sobre um gerenciador de memória virtual. Para isso, será utilizado um material base providenciado para nós de antemão e auxílio de inteligência artificial para desenvolver e entender o raciocínio lógico e o processamento dos dados por trás do sistema, onde os prompts utilizados serão armazenados e documentados para o envio dessa atividade.
